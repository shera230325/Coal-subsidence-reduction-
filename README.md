# AI-Powered Earth Observation for Environmental Crisis Detection

Satellite-based identification of underground coal fire and land subsidence patterns in Jharkhand and Chhattisgarh, India.

**Status: Active Research — Dataset Collection & Exploratory Analysis** — started 2026

## Why this project

Underground coal fires have burned for decades in parts of Jharkhand and Chhattisgarh. Land subsides without warning, and villages have been and continue to be displaced. This project aims to identify these risks earlier, before communities are displaced, using publicly available satellite data.

## What I'm doing

Collecting and analysing publicly available satellite datasets covering these coal mining regions, applying thermal anomaly and interferometric analysis methods to detect underground coal fires and land subsidence patterns. The goal is an early-warning system that can flag environmental collapse indicators before they cause displacement.

This work applies the same constraint-first design logic developed in my prior agricultural AI work (see the soil nutrient CNN repo): build for the person who needs the warning, not just for analytical accuracy.

## Current stage

Dataset collection and exploratory analysis. Initial findings expected by end of 2026.

## Technologies (planned)

Python, satellite imagery analysis (thermal/InSAR), geospatial data tools
